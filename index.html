Krax

This is an example of a REST API for Krax, a highly unofficial library for Krak people search.

You are seeing this page because no search query was specified.

Request format

The only endpoint available is / which accepts the following query string parameters. Providing both lat and lon results in a distance-based search.

  • q (required): text search query
  • limit: max number of results to return
  • lat: latitude as a number, e.g. -12.34
  • lon: longitude as a number

Response format

Results are returned as a JSON array of objects of the following structure:

{
    rank: number
    name: string
    phone: string
    address: string
    zip: string
    city: string
    place: string
    title: string
    email: string
    lat: number
    lon: number
}

Any of these properties may be null depending on the data available.
